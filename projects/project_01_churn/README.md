# Churn Prediction | Predicción de Churn

**Descripción / Description:**  
Este proyecto busca predecir qué clientes tienen mayor riesgo de irse, permitiendo a la empresa tomar decisiones estratégicas de retención.  
This project aims to predict which customers are at higher risk of leaving, allowing the company to take strategic retention actions.

**Caso de uso / Use Case:**  
Optimizar las campañas de retención de clientes y reducir churn.  
Optimize customer retention campaigns and reduce churn.
